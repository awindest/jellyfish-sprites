# Jellyfish Sprites
## An atmospheric phenomenon



Jellyfish sprites are fascinating — those transient luminous events that flash high above thunderstorms in the mesosphere/stratosphere, shaped remarkably like jellyfish with a glowing red body and trailing tendrils. Simulation is on the left and the real thing is on the right in the below image.

## Simulation

### [Website](https://awindest.github.io/jellyfish-sprites/)

Click anywhere on the storm cloud to trigger a lightning stroke and watch a sprite appear above it.

The simulation models the actual physics:

The positive cloud-to-ground lightning bolt fires downward from the storm top
A brief cloud illumination flash follows (the "parent discharge")
The jellyfish sprite appears 80–120ms later in the stratosphere/mesosphere (50–85 km altitude), with its characteristic red/orange diffuse bell and downward-trailing red tendrils
Some sprites include a blue "carrot" streamer shooting upward, and an encircling halo ring — both real features observed in nature.

Atmospheric layer boundaries are labeled with approximate altitudes.

Events also fire automatically every few seconds.

Enjoy!
